# pacifica

150 times faster than "ghq list -p" **but incompatible**.



## Command
```bash
pacifica #default read $HOME/src/**/**
```

## Config
```bash
export PACIFICA_PATH = "/Users/example/src"
pacifica
```

## Installation
```bash
git clone git@github.com:serinuntius/pacifica.git
cargo install --path .
```
